# Tokenization tools

This directory contains the tokenizer and de-tokenizer between MusicXML and proposed score token representation.

- **tokenizer**
  - MusicXML -> Score tokens

- **de-tokenizer**
  - Score tokens -> MusicXML

#### requirements

- **tokenizer**
  - beautifulsoup4 (4.9.3)
  - pretty_midi (0.2.9)
- **de-tokenizer**
  - music21 (6.7.1)
